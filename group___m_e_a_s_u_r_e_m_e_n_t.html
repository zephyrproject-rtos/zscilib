Detailed Description

SI-based measurement representation API.

The measurement API, based around the zsl_measurement struct, is a proof of concept attempt at representing measurements in a concise but unambiguous manner.

It consists of a payload and a zsl_mes_header which describes:

  • A compound measurement type (zsl_mes_type + Extended Type)
  • The SI unit it uses (zsl_mes_unit_si)
  • The specific C type used to represent it in memory (zsl_mes_unit_ctype)
  • An optional scale factor to adjust thee default SI unit scale in +/- 10^n steps (ex. -6 for uA instead of A)
  • Additional meta-data to help interpret the measurement payload
