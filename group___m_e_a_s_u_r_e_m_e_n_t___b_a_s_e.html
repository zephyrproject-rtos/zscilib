enum zsl_mes_type

Base measurement type.

An 8-bit value with a default SI unit type representing a family or class of measurements.

This base type can be further specialised from its default unit/type by making use of an 8-bit extended type value, that extends or refines the type of meassurements that can be represented.

If an extended type is not specified (extended type = 0), the default bases unit will be used.

Memory map:

  • 0 = Undefined
  • 0x10..0x7F = Standard meassuremeent types
  • 0x80..0xEF = Reserved
  • 0xF0..0xFE = User-defined measurement types
